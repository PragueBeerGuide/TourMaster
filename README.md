## Final project of Coding Bootcamp Prague Winter 2023 edition

TourMaster - online tool for booking custom guided tours and events

### Tech stack

- **UI library: React.js**
- **styling: TailwindCSS**
- **backend: Laravel PHP**
- **db: MySQL**
